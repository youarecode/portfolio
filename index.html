Projects

Dementia Prediction

Deployed Large Language Model

2022

This project was a collaboration between UdeSA and me to develop an automatic dementia detector. The model was finetuned from the spanish 'Longformer Narrativa' model. The data for training and testing was given by UdeSA (Centro de Neurociencias Cognitivo), and it is subject to intellectual properties right.

Note that the model was trained for spanish only speakers. Otherwise i can't guaranteed the predicted accuracy.

The model weights 500mb for alzheimer disease, and 500mb for FTDbv disease. So both of them won't fit inside a free web host. So the way i found to share to you is through a tunneling to my personal computer.

To try this out, please ask me to run the server. And visit the following link
